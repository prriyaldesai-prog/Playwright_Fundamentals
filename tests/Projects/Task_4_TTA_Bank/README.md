# 🎭 Playwright Automation Framework

A modern, high-performance web automation framework built with **Playwright**, featuring a premium **Custom TTA Reporter** for detailed execution visibility.

## 🚀 Key Features

- **Custom TTA Reporter**: Real-time HTML reports with detailed step-by-step execution.
- **Visual Evidence**: Automatic screenshots and video recordings for every test run.
- **Multiple Browsers**: Support for Chromium, Firefox, and WebKit.
- **Trace Viewer**: Deep dive into test failures with Playwright's trace files.
- **Allure Integration**: Integrated with Allure Reporting for advanced analytics.

## 🛠️ Setup and Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd "Playwright Fundamentals"
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Install Playwright browsers:**
   ```bash
   npx playwright install
   ```

---

## 🏃 Running Tests

To run all tests:
```bash
npx playwright test
```

To run a specific test file:
```bash
npx playwright test tests/Projects/Task_4_TTA_Bank/Task_TTA_Bank.spec.ts
```

To run tests with the UI Mode:
```bash
npx playwright test --ui
```

## 📈 Viewing Reports

### Custom TTA Report
Open the following file in your browser:
`tta-report/index.html`

### Allure Report
1. Generate the report:
   ```bash
   npx allure generate allure-results --clean
   ```
2. Open the report:
   ```bash
   npx allure open allure-report
   ```

## 📁 Project Structure

- `tests/`: Contains all the test specifications.
- `playwright.config.ts`: Main configuration for Playwright.
- `tta-report/`: Custom reports generated by `CustomTTAReporter`.
- `allure-results/`: Data for Allure reporting.
